## Linear probability model

- We still care about the conditional mean
- If our Y is a 0 or 1, dummy variable, the expected value is also:
  - $E[y|x]=P(Y=1|x)=\beta_0+\beta_1x_1+\beta_2x_2+...$
  - We're making an assumption that all the associated variables come in a linear form
- We can predict the probability of an event happening, use the model to classify y's into 0 or 1 (classification), and find association with other variables
